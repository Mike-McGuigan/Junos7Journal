# Project Notes — Juno's 7 Mediterranean Journal

## Current milestone

Version: 1.0.0
Release: Juno's 7 Companion Pages

## Design principles

- The heart of the site is Cameron and Sophie's season, not yacht tracking for its own sake.
- Do not identify private guests, children, or other crew unless explicitly confirmed and appropriate.
- Do not state that SETE appears in a photograph unless it is clearly visible.
- Keep `site/` as the development site and `docs/` as the GitHub Pages output.
- Use the ZIP + clear commands workflow for safety.

## Navigation

Core pages:
- `index.html` — Journal homepage
- `voyage.html` — Interactive route map
- `about.html` — About Juno's 7
- `crew.html` — Meet the Crew

## Sources for yacht facts

Use public sources conservatively. Public sources differ slightly on crew/guest/cabin figures, so the site uses ranges where appropriate.
