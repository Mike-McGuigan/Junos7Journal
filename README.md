# Junos7Journal
Juno's 7 Mediterranean Journal 2026

> *Don't document the yacht. Document the memories.*

---

## About

This repository contains the source for an interactive journal documenting the 2026 Mediterranean season of the superyacht **Juno's 7**.

While the project follows the yacht's route through the Mediterranean, it is primarily a record of **Cameron and Sophie's first season working together aboard the same yacht**.

The aim is to preserve the journey through photographs, videos, maps, AIS tracking, journal entries and stories, eventually producing both:

- 🌐 An interactive digital journal
- 📖 A professionally printed coffee-table book

This is intended to become a lasting family keepsake rather than simply a vessel tracker.

---

# Project Vision

Most vessel tracking projects answer one question:

> "Where is the yacht?"

This project answers a much more important one:

> "What was it like to be there?"

Rather than simply recording positions on a chart, the journal combines:

- Routes
- Anchorages
- Photographs
- Videos
- Local history
- Notable neighbouring yachts
- Crew memories
- Daily journal entries
- Statistics
- Interactive maps

to tell the story of an entire Mediterranean season.

---

# Guiding Principle

> **Don't document the yacht. Document the memories.**

Everything in this project exists to support that idea.

The AIS positions, photographs, videos and maps are not the destination—they are the framework that helps preserve the memories made along the way.

# Design Philosophy

The project deliberately avoids looking like a traditional website.

Instead, it aims to feel like opening a beautifully produced travel journal.

The design is inspired by:

- National Geographic
- Boat International
- Apple product photography
- Premium travel books

The emphasis is on:

- Large photography
- Elegant typography
- Calm navigation
- Minimal interface
- Storytelling
